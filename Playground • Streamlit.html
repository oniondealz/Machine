Playground

Try out a limited version of Streamlit right in your browser.

Just edit the code below and the app on the right updates automatically.

For the real thing, install our Python library.

Examples
import streamlit as st

st.title("I am Wahab")
st.header("Head")
st.subheader("Sup")
st.write("Writing...")

st.button("Submit")
st.radio("Choose an option", [1,2,3])
st.text_input("Enter text here")
st.slider("Pick a number",1,100)
st.selectbox("Choose: ",["A","B","C"])
st.checkbox("Check me")
st.image("https://images.pexels.com/photos/3861976/pexels-photo-3861976.jpeg?auto=compress&cs=tinysrgb&w=1260&h=750&dpr=1")
